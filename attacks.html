Différentes attaques informatiques

De nombreuses attaques peuvent être effectuées à l'encontre d'un système informatique, nous allons en voir quelques-unes

  • Phishing : c'est une attaque qui consiste à envoyer des e-mails ou des messages de phishing pour tenter de voler des informations personnelles, comme des mots de passe ou des données de carte de crédit. On peut utiliser une sandbox pour exécuter du code pouvait potentiellement affecter le système.
  • DDoS (Distributed Denial of Service) : c'est une attaque qui consiste à surcharger un site Web ou un service en ligne en envoyant un grand nombre de requêtes simultanées, rendant ainsi le site ou le service inaccessible aux utilisateurs légitimes. On peut notamment utiliser un botnet.
  • Les attaques par Drive by Download : la personne malveillante insère un script malveillant dans le code HTTP ou bien PHP d'une page qui permet d'installer un logiciel malveillant. Cette attaque est particulièrement viscieuse car elle ne nécessite par d'intéraction de la part de l'utilisateur. Pour s'en protéger, il faut effectuer les mises à jour de sécurité du système d'exploitation et des logiciels.
  • Les attaques par mot de passe :
    • Les attaques dites par force brute : on tente des mots de passe de façon aléatoire de manière à trouver le mot de passe utilisé
    • Les attaques par disctionnaire : on copie un fichier chiffré contenant les mots de passe, on applique le même chiffrement à un disctionnaire de mots de passe couramment utilisés, puis on compare les résultats pour trouver le mot de passe utilisé
  • L'attaque "man in the middle" : la personne malveillante grâce à des outils adaptés va intercepter le trafic réseau entre le client et le serveur
